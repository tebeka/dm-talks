A Tour Of Scientific Python

Miki Tebeka <miki.tebeka@demandmedia.com>

Why?

We have a lot of data that we want to play. Often we can run algorithms on a single core. However, it takes a long time to develop new algorithms for Hadoop.

We found it’s faster to run an aggregation on Hadoop, then explore the data with Python. Once there’s something that works, we can convert it to Mahout and friends.

Also, you can use Scientific Python for other tasks. For example, we used pandas to compare database after code changes.

import matplotlib.pylab as plt
import numpy as np

# school.txt has comma separated lines: score1,score2,admitted
data = np.loadtxt('src/school.txt', delimiter=',')
passed = data[data[:,2]==1]  # Select admitted
failed = data[data[:,2]==0]  # Select not admitted

fig = plt.figure()
ax = fig.add_subplot(111)
ax.scatter(passed[:,0], passed[:,1],  # Slice xs and ys
           color='blue', marker='+', label='Admitted')
ax.scatter(failed[:,0], failed[:,1],  # Slice xs and ys
           color='red', marker='o', label='Not admitted')
ax.set_xlabel('Exam 1 score')
ax.set_ylabel('Exam 2 score')
plt.legend()
plt.show()

import numpy as np
import matplotlib.pyplot as plt
from sklearn import cluster
# Create random matrix of 1000x2
data = np.random.rand(1000, 2)
# Run k-means
km = cluster.KMeans(k=5)
km.fit(data)
# Plot the results
fig = plt.figure()
ax = fig.add_subplot(111)
# Data
ax.scatter(data[:,0], data[:,1])
# centers
centers = km.cluster_centers_
ax.scatter(centers[:,0], centers[:,1], color='red', marker='s')
plt.show()

import db
from pandas import DataFrame
import matplotlib.pyplot as plt

pre_cursor = db.get('pre')   # Dummy load
post_cursor = db.get('post')

pre = DataFrame.from_records([row for row in pre_cursor])  # 
post = DataFrame.from_records([row for row in post_cursor])

diff = ((pre - post)/pre * 100).abs()  # Diff in %
print(diff.describe())  # Show stats
diff['revenue'].hist()  # Plot histogram
plt.show()

import nltk

tokens = nltk.word_tokenize(sentence)
